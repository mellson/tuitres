# TUITRIS

Tetris TUI game built with Ink and React.

## Installation

```bash
npm install
npm run build
```

## Usage

```bash
npm start
```

Or run directly:

```bash
node dist/index.js
```

## Controls

- **←→** Move left/right
- **↑** Rotate piece
- **↓** Soft drop
- **Space** Hard drop
- **P** Pause/unpause
- **R** Restart game
- **Q** Quit

## Features

- Classic Tetris gameplay
- All 7 standard tetromino pieces (I, O, T, S, Z, J, L)
- Line clearing and scoring
- Progressive difficulty (speed increases with level)
- High score system (top 10 saved to highscore.json)
- Responsive to terminal size
- Next piece preview

## Requirements

Minimum terminal size: 50x25
